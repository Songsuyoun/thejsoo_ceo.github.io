📅 주휴수당 지급 조건
① 해당 주 15시간 이상 근무
② 일반직원 — 전체 영업일 개근
   결근 있으면 주휴 없음
③ 유가경 · 오윤지 (계약직)
   주 15시간 미만 근무 → 주휴수당 미발생
   하루 3시간 × 4일 = 12시간 → 15시간 미충족

주휴수당 = 주간 총 근무시간 ÷ 5 × 시급(10,320원)
🛠️ 사용 방법
1. 출근시간 직접 수정
  → 출근(보정) 셀 클릭 → 시간 입력 → 저장
    Enter 저장 / Esc 취소

2. 결근 의심 처리
  → 다른 직원은 출근했는데 해당 직원만 없는 날 자동 감지
  → 결근 맞음 선택 시 해당 주 주휴수당 없음
  → 아님 선택 시 출근으로 인정, 주휴 정상 계산
  → 선택 후 [재계산] 버튼 클릭 필수

3. 재계산
  → 결근 처리 변경 후 반드시 [🔄 재계산] 클릭
  → 출근시간 수정은 즉시 자동 반영

4. 다운로드
  → 📥 엑셀 — 상세 근무 내역 + 급여 요약표
  → 🖼️ 이미지 — 직원별 카톡 전송용 PNG (ZIP)
※ 근무시간은 10분 단위 내림 처리됩니다 (예: 3시간 37분 → 3시간 30분)  |  시급: 10,320원
⚠️ 결근 의심 확인 필요  결근맞음 선택 시 해당 주 주휴수당 없음 처리
※ 확인 후 [재계산] 버튼 클릭 시 반영됩니다
